Welcome to the repository for my personal portfolio website! This site showcases my projects, skills, education, and professional journey as a developer. It serves as a central hub for anyone looking to learn more about my work or get in touch.

## 🚀 Live Site

Check out the live version here: **[https://hiarchit.online](https://hiarchit.online)**

## 🧐 About The Project

This portfolio is designed to be a clean, responsive, and interactive representation of my resume and work. It highlights my technical stack and provides direct links to my projects and social profiles.

**Key Features:**

- **Responsive Design:** Fully optimized for desktop, tablet, and mobile viewing.
- **Project Showcase:** Dedicated cards/sections for individual projects with GitHub and deployment links.
- **About Me:** A summary of my background, education, and interests.
- **Contact Section:** Easy ways to connect with me via email or social media.
- **Smooth Navigation:** Intuitive UI for a seamless user experience.

## 🛠️ Technologies Used

This project was built using the following technologies:

- ![Angular](https://img.shields.io/badge/angular-%23DD0031.svg?style=flat&logo=angular&logoColor=white) **Angular** - Robust framework for the main application.
- ![TypeScript](https://img.shields.io/badge/typescript-%23007ACC.svg?style=flat&logo=typescript&logoColor=white) **TypeScript** - Typed JavaScript for scalable logic.
- ![SCSS](https://img.shields.io/badge/SASS-hotpink.svg?style=flat&logo=SASS&logoColor=white) **SCSS** - For advanced styling and layout.
- ![GSAP](https://img.shields.io/badge/GSAP-green.svg?style=flat) **GSAP** - For smooth, high-end animations.
- **Render** - For hosting and deployment.
